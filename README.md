# Notification System

## Install

Checkout the project, then:

```bash
pnpm i
```

then point your browser to: 

```http://localhost:5173/```

([link](http://localhost:5173/))


## Run it

in one terminal run

```bash
docker compose up
```

in another one run: 

```bash
pnpm dev
```




## Test

while you are runing ```pnpm dev``` you can run tests and typecheck with:

```bash
pnpm all
```

You can also run them separatly with

```bash
pnpm typecheck
```

or

```bash
pnpm test
```
